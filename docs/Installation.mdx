# Installation

You can install Cakebase easily with [npm](https://www.npmjs.com/), [yarn](https://classic.yarnpkg.com/en/) or [pnpm](https://pnpm.io/pnpm-cli).

<Tabs

  ]}>
  <TabItem value="npm">
```bash
npm i cakebase
```
  </TabItem>

  <TabItem value="yarn">
```bash
yarn add cakebase
```
  </TabItem>

  <TabItem value="pnpm">
```bash
pnpm add cakebase
```
  </TabItem>
